# 研报策略：红利低波的增强方案

> 来源：国盛证券金融工程研究，沈芷琦、刘富兵、赵博文，2023-08-16，《“薪火”量化分析系列研究（三）：红利低波的增强方案》  
> 研究对象：红利低波指数（H30269.CSI）及其全收益指数  
> 策略类型：横截面选股、指数增强、择时、混合策略

---

## 1. 研报核心结论

| 策略 | 核心改动 | 年化收益 | 年化波动 | 信息比率 | 月度胜率 | 最大回撤 |
|---|---:|---:|---:|---:|---:|---:|
| 红利低波全收益指数 | 原始基准 | 13.22% | 20.10% | 0.66 | 61.95% | 34.48% |
| 红利低波月调组合 | 将年频调样改为月频 | 16.68% | 17.94% | 0.93 | 61.95% | 26.64% |
| 高频波动率月调组合 | 月频调样 + 高频波动率替代传统波动率 | 20.92% | 21.74% | 0.96 | 60.18% | 29.05% |
| 估值差择时：空仓 | 看空信号后下月空仓 | 18.45% | 18.28% | 1.01 | 69.91% | 21.12% |
| 估值差择时：卖空 | 看空信号后下月卖空红利低波全收益指数 | 23.20% | 19.44% | 1.19 | 66.37% | 21.12% |

研报认为红利低波指数原始编制方案的主要问题是：

1. **调样周期过长**：原指数每年 12 月调样一次，无法及时剔除新一年中波动率显著上升的股票。
2. **传统波动率因子不稳定**：原指数使用过去一年日收益率标准差作为低波代理变量，该因子在红利股票池中的多空对冲效果较弱。

---

## 2. 原始红利低波指数编制方案

### 2.1 调样频率

- 每年 12 月第二个周五的下一个交易日进行成份调整。

### 2.2 样本空间

在全部 A 股中，选择同时满足以下条件的股票：

1. 过去三年连续分红，且每年税后现金股息率大于 0。
2. 过去一年日均总市值排名在全部 A 股前 80%。
3. 过去一年日均成交额排名在全部 A 股前 80%。

### 2.3 选样流程

在样本空间中：

1. 计算过去一年的红利支付率、过去三年的每股股利增长率。
2. 剔除：
   - 红利支付率过高的股票，即红利支付率降序排名前 5%；
   - 红利支付率为负的股票；
   - 每股股利增长率为负的股票。
3. 对剩余股票计算过去三年平均税后现金股息率，降序排序，选前 75 只。
4. 在上述 75 只股票中，计算过去一年波动率，升序排序，选前 50 只。
5. 以股息率加权构建指数。

### 2.4 权重方式

- 权重指标：股息率。
- 具体是否存在单股权重上限：研报正文未明确说明，复刻时需查指数编制细则补充。

## 3. 策略一：红利低波月调组合

### 3.1 策略类型

- 类型：横截面选股策略（cross-sectional strategy）
- 目标：复刻红利低波指数逻辑，但将调样周期由年频缩短至月频。

### 3.2 股票池

沿用原始红利低波指数样本空间：

1. 全部 A 股。
2. 过去三年连续分红，且每年税后现金股息率大于 0。
3. 过去一年日均总市值排名前 80%。
4. 过去一年日均成交额排名前 80%。

### 3.3 选股步骤

每月末执行：

1. 在样本空间内，计算：
   - 过去一年红利支付率；
   - 过去三年每股股利增长率。
2. 剔除：
   - 红利支付率降序排名前 5%；
   - 红利支付率为负；
   - 每股股利增长率为负。
3. 筛选后的股票池称为**红利股票池**。
4. 对红利股票池计算过去三年平均税后现金股息率。
5. 按过去三年平均税后现金股息率降序排序，选前 150 只。
6. 在前 150 只股票中，计算过去一个月传统波动率。
7. 按过去一个月传统波动率升序排序，选前 50 只。
8. 按股息率加权。

### 3.4 因子定义

#### 3.4.1 传统波动率因子

```text
traditional_volatility_i,t = std(daily_return_i, t-lookback:t)
```

研报在月调组合中使用过去一个月波动率；原始指数使用过去一年波动率。

### 3.5 回测结果

| 指标 | 红利低波月调组合 | 红利低波全收益指数 |
|---|---:|---:|
| 年化收益 | 16.68% | 13.22% |
| 年化波动 | 17.94% | 20.10% |
| 信息比率 | 0.93 | 0.66 |
| 月度胜率 | 61.95% | 61.95% |
| 最大回撤 | 26.64% | 34.48% |

### 3.6 研报指出的缺陷

虽然全样本跑赢红利低波全收益指数，但 2016、2017、2018 年连续跑输，说明仅缩短调样周期无法完全解决传统低波因子的稳定性问题。

## 4. 策略二：基于高频波动率因子的红利低波月调组合

### 4.1 策略类型

- 类型：横截面选股策略（cross-sectional strategy）
- 目标：在红利低波月调组合基础上，用更稳定的高频波动率因子替代传统波动率因子。

### 4.2 高频波动率因子定义

研报使用国盛金工的 `minute_return_volatility` 因子。

#### 第一步：计算日内分钟收益波动率

每个交易日，对每只股票计算分钟涨跌幅标准差：

```text
vol_daily_i,d = std(minute_return_i,d,m)
```

其中：

- `i`：股票；
- `d`：交易日；
- `m`：日内分钟 bar；
- `minute_return_i,d,m`：股票 i 在交易日 d 的分钟收益率。

#### 第二步：计算过去 20 个交易日 `vol_daily` 的变异系数

每月末回看过去 20 个交易日：

```text
cv_vol_i,t = std(vol_daily_i,t-19:t) / mean(vol_daily_i,t-19:t)
```

#### 第三步：横截面市值中性化

对 `cv_vol_i,t` 做横截面市值中性化处理，得到：

```text
minute_return_volatility_i,t = residual(cv_vol_i,t ~ log_market_cap_i,t)
```

> 复刻时建议使用普通最小二乘回归（OLS）按月截面残差化。

### 4.3 因子方向

- 高频波动率越低越好。
- 排序方向：升序。
- 选股：选择因子值最低的股票。

### 4.4 因子分组效果

#### 全体 A 股

| 指标 | 传统波动率 | 高频波动率 |
|---|---:|---:|
| 年化收益 | 9.32% | 23.74% |
| 年化波动 | 19.62% | 7.26% |
| 信息比率 | 0.48 | 3.27 |
| 月度胜率 | 58.41% | 83.19% |
| 最大回撤 | 31.89% | 3.82% |
| IC 均值 | -0.039 | -0.057 |
| 年化 ICIR | -0.927 | -3.199 |
| RankIC 均值 | -0.079 | -0.082 |
| 年化 RankICIR | -1.579 | -4.389 |

#### 红利股票池

| 指标 | 传统波动率 | 高频波动率 |
|---|---:|---:|
| 年化收益 | 3.64% | 16.31% |
| 年化波动 | 18.34% | 8.35% |
| 信息比率 | 0.20 | 1.95 |
| 月度胜率 | 52.21% | 71.68% |
| 最大回撤 | 31.75% | 10.79% |
| IC 均值 | -0.025 | -0.049 |
| 年化 ICIR | -0.526 | -2.431 |
| RankIC 均值 | -0.057 | -0.067 |
| 年化 RankICIR | -1.073 | -2.726 |

### 4.5 组合构建流程

每月末执行：

1. 沿用原始样本空间。
2. 剔除：
   - 红利支付率降序排名前 5%；
   - 红利支付率为负；
   - 每股股利增长率为负。
3. 对剩余股票计算过去三年平均税后现金股息率。
4. 按过去三年平均税后现金股息率降序排序，选前 150 只。
5. 在前 150 只股票中，计算过去一个月高频波动率因子。
6. 按高频波动率因子升序排序，选前 50 只。
7. 按股息率加权构建组合。

### 4.6 回测结果

| 指标 | 高频波动率月调组合 | 普通月调组合 | 红利低波全收益指数 |
|---|---:|---:|---:|
| 年化收益 | 20.92% | 16.68% | 13.22% |
| 年化波动 | 21.74% | 17.94% | 20.10% |
| 信息比率 | 0.96 | 0.93 | 0.66 |
| 月度胜率 | 60.18% | 61.95% | 61.95% |
| 最大回撤 | 29.05% | 26.64% | 34.48% |

## 5. 策略三：基于估值差的红利低波择时

### 5.1 策略类型

- 类型：择时策略（timing strategy）
- 目标：根据红利低波指数成份股与非成份股的相对估值，判断红利低波策略是否可能失效。

### 5.2 估值指标

使用 BP，即市净率倒数：

```text
BP = 1 / PB
```

### 5.3 指标构建流程

每月末执行：

1. 对红利低波指数成份股：
   - 计算每只成份股的 BP；
   - 剔除 BP 最高的 10% 和最低的 10%；
   - 对剩余股票计算 BP 均值，记为 `BP_low`。
2. 对非红利低波指数成份股：
   - 计算每只股票的 BP；
   - 剔除 BP 最高的 10% 和最低的 10%；
   - 对剩余股票计算 BP 均值，记为 `BP_high`。
3. 计算估值差：

```text
BP_spread_t = BP_low_t - BP_high_t
```

### 5.4 信号规则

参考布林带（Bollinger Band）思想：

```text
lower_band_t = mean(BP_spread_{t-5:t}) - 1.5 * std(BP_spread_{t-5:t})
```

当月末估值差下穿过去 6 个月估值差的下轨时，发出看空信号：

```text
bearish_signal_t = BP_spread_t < lower_band_t
```

研报解释：

- `BP_spread` 明显下降，代表红利低波指数成份股相对非成份股估值偏贵；
- 低波策略未来可能反转或失效；
- 下月应低配或规避红利低波指数。

### 5.5 持仓规则

研报测试两种情形：

#### 情形 A：看空后空仓

```text
if bearish_signal_t:
    next_month_return = 0
else:
    next_month_return = dividend_low_vol_total_return_next_month
```

#### 情形 B：看空后卖空

```text
if bearish_signal_t:
    next_month_return = -dividend_low_vol_total_return_next_month
else:
    next_month_return = dividend_low_vol_total_return_next_month
```

### 5.6 回测结果

| 指标 | 红利低波全收益指数 | 择时净值：空仓 | 择时净值：卖空 |
|---|---:|---:|---:|
| 年化收益 | 13.22% | 18.45% | 23.20% |
| 年化波动 | 20.10% | 18.28% | 19.44% |
| 信息比率 | 0.66 | 1.01 | 1.19 |
| 月度胜率 | 61.95% | 69.91% | 66.37% |
| 最大回撤 | 34.48% | 21.12% | 21.12% |
| 最大回撤时间段 | 201506-201602 | 201508-201602 | 201508-201602 |

## 6. 附录指标计算细节

| 数据名称 | 数据说明 | 计算方法 |
|---|---|---|
| 平均市值 | 过去一年平均市值 | 过去 240 个交易日总市值平均值 |
| 平均成交额 | 过去一年平均成交额 | 过去 240 个交易日成交额平均值 |
| 过去三年是否连续分红 | 1 代表连续分红，0 代表未连续分红 | 根据当前月份确认回看过去三年年份；只有 12 月末才使用当年数据 |
| 股利支付率 | 当月末股利支付率 | 股息率 TTM * PE TTM |
| 每股股利增长率 | 过去三年每股股利增长率 | 过去三年每股股利回归斜率；若当月股票已经分红，则使用最新数据，否则不考虑今年最新数据 |
| 过去三年平均股息率 | 过去三年平均股息率 | 过去 720 个交易日股息率 TTM 平均值 |

> 项目复刻口径：股利支付率按正文财务恒等式使用“股息率 TTM × PE TTM”。股票池构建以标准分红数据集的预案公告日字段 `announce_date` 不晚于调仓日为可用条件，不等待实际派息。当前行情数据不保留停牌日，因此过去 240 个交易日暂按每只股票最近 240 条有效行情近似；具体上游来源及字段映射以 `configs/datasets.yaml` 为准。
